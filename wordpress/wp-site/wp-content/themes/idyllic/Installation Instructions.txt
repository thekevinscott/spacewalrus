Install Info

- Activate plugins
- Activate theme

Changing Flickr to point to your images
---------------------------------------

- Open sidebar.php using WP or a text editor like notepad

- Find the Flickr script

- Change "tag=art" to "tag=your tag" "your tag" being your own unique tag
